# Merula
A simple, working native Android app built with Kotlin and Jetpack - Compose to practice state and layouts.

Simple but working app for randome daily affirmations.

Technical Overview:
* Language -> Kotlin
* UI Framework -> Jetpack - Compose
* State Management -> simple mutable state tracking
* Target -> Build as practice
* Images generated with Canva

Key Quality Features 
*Crash-Free Desing -> Validated state to ensure no rendering freezes during text changes
*Responsive Layout -> Built with a flexible Compose layout to adjust cleanly across different device screen sizes
*Clean Code -> Standart Kotlin Conventions applied for optimal code readability
